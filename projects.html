Projetos e Implementações
Galeria de projetos focada em sistemas de baixo nível, lógica matemática e renderização gráfica. Cada repositório representa um marco na minha curva de aprendizado.
Desenvolvimento de Engine Gráfica (Core)
Projeto principal focado na criação de uma fundação sólida para renderização 3D. Inclui a abstração de janelas, gerenciamento de contexto OpenGL e sistemas de carregamento de recursos.
// Features Implementadas
- Integração GLFW / GLAD
- Sistema de Classes para Vertex/Fragment Shaders
- Gerenciamento de Input (Teclado/Mouse)
Tecnologias: C++ Moderno, OpenGL 3.3, Visual Studio.
Projetos de Consolidação C++
Conjunto de aplicações desenvolvidas para aplicar conceitos de POO e algoritmos. Inclui lógica de jogos clássicos como Snake e Termo, focando em estruturação de código limpo.
Destaque: Implementação de matrizes dinâmicas e sistemas de pontuação persistente.
Laboratório de Design e Assets
Criação de elementos visuais e prototipagem de interfaces para aplicações gráficas. Foco na exportação de vetores e texturas para integração via código.
Ferramentas: CorelDRAW para vetores e Canva para UI/UX de baixa fidelidade.
"O código é a ferramenta, o resultado é a engenharia."
